vnet-vagrant
============

OpenVNet installer over Vagrant

Requirements
============

* virtualbox
* vagrant >= 1.6.5

Usage
=====

clone the repository.

```
git clone https://github.com/akry/vnet-vagrant.git
cd vnet-vagrant
```

launch the vna node.

```
vagrant up
```

login the node by ssh

```
vagrant ssh vnmgr_vna1
```
